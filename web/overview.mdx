The GAM web admin panel is a browser-based management interface for UTL staff and administrators. It is built with **Bootstrap 5**, uses the **Inter** typeface, and communicates with the Java EE REST API backend at `http://localhost:8080/IntegradoraII_Web/api`.

After login, administrators are directed to the dashboard where they can navigate to any module using the persistent left sidebar.

<Note>
  All pages require an active session. Session data (`id_usuario`, `id_rol`, `nombre_usuario`, `matricula`) is stored in `localStorage`. Clicking the avatar icon in the topbar clears the session and returns the user to the login page.
</Note>

## Browser requirements

The panel relies on standard browser APIs (`fetch`, `localStorage`, CSS Grid/Flexbox). Any modern evergreen browser is supported:

| Browser | Minimum version |
|---|---|
| Chrome / Edge | 88+ |
| Firefox | 85+ |
| Safari | 14+ |
