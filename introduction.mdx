# GAM — Gestor Académico para Mobiliario

GAM (Academic Furniture Manager) is the institutional resource management platform for **UTL (Universidad Tecnológica de León)**. It provides administrators with full control over classrooms, furniture inventory, equipment loans, and room reservations — all from a unified web panel and a cross-platform mobile app.

## System components

GAM is composed of three main layers:

| Layer | Technology | Purpose |
|---|---|---|
| **Backend REST API** | Java EE (JAX-RS) + MySQL | Business logic and data persistence |
| **Web Admin Panel** | HTML / JavaScript / Bootstrap 5 | Browser-based admin interface |
| **Mobile App** | Ionic 8 + Angular 20 + Capacitor 8 | Cross-platform iOS/Android app |

## Who uses GAM

- **Administrators** — manage all system resources via the web panel or mobile app
- **Staff** — create reservations and request equipment loans
- **Institution** — UTL uses GAM to maintain full visibility over campus resource utilization

<Note>
  GAM is an internal institutional system. Access requires a valid UTL matricula and password issued by an administrator.
</Note>
